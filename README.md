# AI-SmartHealth-Assistant
The AI-Powered Smart Health Assistant is a unified platform that delivers real-time first aid guidance, early disease prediction, and mental health analysis. By leveraging AI, it reduces response time, improves early diagnosis, and empowers individuals—especially in rural and high-risk areas—to proactively manage their health and save lives.


# AI Smart Health Assistant

A modern AI-powered web app for:
- First Aid Assistance
- Disease Prediction
- Mental Health Analysis

## Tech Stack
- React + Tailwind
- Node.js + Express

## Features
- Voice-based mental health detection
- Emergency first aid guidance
- Mock AI predictions
